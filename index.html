VERITAS – Plataforma Corporativa de Compliance

O projeto VERITAS é uma aplicação web moderna desenvolvida com
Next.js 14 e TypeScript, desenhada para atender às necessidades de um programa corporativo de compliance. A plataforma está preparada para ser implantada em ambientes escaláveis, como o Google Cloud, e segue as melhores práticas de desenvolvimento e governança.

Funcionalidades Principais

  • Autenticação: tela de login segura com validação de credenciais.
  • Dashboard: visão geral com indicadores de riscos ativos, planos em andamento e nível geral de compliance.
  • Matriz de Riscos: cadastro e gestão de riscos identificados, com categorização, probabilidade, impacto e responsáveis.
  • Análise Avançada: gráfico de barras que demonstra a distribuição dos níveis de riscos e relatório de inconsistências.
  • Frameworks de Compliance: gerenciamento de frameworks (ISO, COSO etc.) e acompanhamento de requisitos.
  • Planos de Ação e Contingências: criação de planos detalhados para mitigação de riscos, com etapas, custos e responsáveis.
  • Lixeira: área para arquivamento e exclusão definitiva de riscos e planos.

Estrutura do Projeto

O código está organizado em módulos dentro do diretório src:

  • app/: contém as rotas da aplicação, incluindo login, dashboard, matriz de riscos, frameworks, planos de ação e lixeira.
  • components/: reúne componentes reutilizáveis, como formulários, tabelas, gráficos e relatórios.
  • contexts/: define contextos de autenticação e dados, responsáveis pelo estado global e persistência.
  • lib/: contém definições de tipos, utilitários, constantes iniciais e configuração do Firebase/Firestore.

Instalação e Execução

Para instalar e executar o VERITAS localmente, siga estas etapas:

  1. Clone o repositório e navegue até a pasta veritas.
  2. Instale as dependências com npm install.
  3. Crie um arquivo .env.local com as credenciais do Firebase (se utilizar Firestore/Storage).
  4. Execute npm run dev para iniciar o servidor de desenvolvimento.
  5. Acesse http://localhost:3000 no navegador para utilizar a aplicação.

Considerações Finais

Esta aplicação foi concebida para evoluir conforme a maturidade do programa de compliance, permitindo migração futura de dados para o Firestore e armazenamento de arquivos no Google Cloud Storage. O design responsivo e a arquitetura modular facilitam a manutenção e a escalabilidade do sistema.
